# テーマカラーの青緑色（Teal）への統合完了

システム全体のテーマカラーを従来のブルーから、ご要望いただいた青緑色（Teal）に刷新しました。

## 実施内容

### 1. グローバルスタイルの調整
- `src/app/globals.css` のサイドバーのアクティブな残像効果（`.nav-active-glow`）を Teal 系に調整しました。

### 2. レイアウトコンポーネントの刷新
- **サイドバー (`SidebarNav.tsx`)**: 
  - 背景を `teal-950`、アクセントを `teal-400` に変更。
  - ホバーやアクティブ状態の配色も Teal 系統で統一。
- **ヘッダー (`Header.tsx`)**:
  - タイトル左側のアクセントバーとユーザーアイコンを `teal-700` に変更。

### 3. 各機能ページの配色統一
- **ダッシュボード (`src/app/page.tsx`)**: 
  - メインボタン（入庫・出庫）や検索窓のフォーカスリング、テーブルのホバー色を Teal に変更。
- **マスタ管理 (`src/app/master/page.tsx`)**:
  - 管理カードの配色を Teal で統一。
- **製品マスタ (`src/app/master/products/page.tsx`)**:
  - 新規登録・編集ボタン、テーブルのホバー配色を Teal に変更。
- **倉庫マスタ (`src/app/master/warehouses/page.tsx`)**:
  - 従来の Emerald 系統から Teal 系統へ微調整し、アプリ全体とのトーンを統合。
