# ダッシュボード検索条件の拡張完了

ダッシュボードに「カテゴリ」と「保管倉庫」による絞り込み機能を追加しました。

## 実施内容

### 1. 状態管理の追加
`src/app/page.tsx` に以下の状態を追加しました。
- `filterCategory`: 製品カテゴリによるフィルタ
- `filterWarehouse`: 保管倉庫IDによるフィルタ

### 2. UIの更新
ツールバーの検索窓の横に、新しいドロップダウンメニューを追加しました。
- **カテゴリ選択**: 「すべて」「通常品」「生鮮食品」「危険物」から選択可能。
- **倉庫選択**: マスターデータから取得した倉庫一覧から選択可能。

### 3. ロジックの拡張
`filteredInventory` の `useMemo` ロジックを、テキスト検索、カテゴリ、倉庫のすべての条件が一致するように更新しました。

## 確認内容

### フィルタリングの動作
- **テキスト検索**: 製品名や倉庫名での部分一致検索。
- **カテゴリフィルタ**: `inv.category` との完全一致による絞り込み。
- **倉庫フィルタ**: `inv.warehouseId` との完全一致による絞り込み。
