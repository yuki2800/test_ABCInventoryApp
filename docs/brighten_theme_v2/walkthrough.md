# テーマカラーの明色化とプライマリ変数移行の完了

ご要望に合わせて、システム全体のテーマカラーをより明るい青緑色（Cyan）に変更しました。また、将来的な調整を容易にするため、コード内で直接色を指定するのではなく、`primary` というカスタム変数を使用するように刷新しました。

## 実施内容

### 1. プライマリカラーの定義 (`globals.css`)
- `src/app/globals.css` の `@theme inline` ブロックにて、`primary` カラーを定義。
- 現在は `cyan` パレットにマッピングされており、以前の `teal` よりも明るく、彩度の高い青緑色になっています。
- これにより、今後の微調整は `globals.css` の定義を書き換えるだけで、全コンポーネントに反映されます。

### 2. 全ページ・コンポーネントの構造化
- 以下の全てのファイルについて、色指定を `teal-` や `blue-` から `primary-` に置換しました。
  - `src/components/SidebarNav.tsx`
  - `src/components/Header.tsx`
  - `src/app/page.tsx`
  - `src/app/master/page.tsx`
  - `src/app/master/products/page.tsx`
  - `src/app/master/warehouses/page.tsx`

## 調整のポイント
- **鮮快な印象**: シアンを採用したことで、システム全体の清潔感とモダンな印象が向上しました。
- **ダークサイドバーの維持**: サイドバーの背景は `primary-950`（非常に濃いシアン）とすることで、プレミアムな重厚感を保ちつつ、テーマカラーとの親和性を高めています。
- **アクティブグロウ**: サイドバーのアクティブ項目の残像効果も、新色に合わせて調整されています。
